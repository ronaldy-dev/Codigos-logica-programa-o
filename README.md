# Codigos-logica-programação
# 🐍 Lógica de Programação com Python

Este repositório foi criado para documentar e organizar a minha evolução nos estudos de **Lógica de Programação**. Aqui, deposito todos os exercícios, desafios e projetos práticos que desenvolvo utilizando a linguagem Python.

## 🎯 Objetivo
O foco principal é fortalecer a base de algoritmos, estrutura de dados e pensamento computacional, servindo como um registro da minha jornada acadêmica em Engenharia de Software.

## 🛠️ O que você encontrará aqui?
* **Variáveis e Operadores:** Exercícios iniciais de entrada e saída de dados.
* **Estruturas Condicionais:** Lógica de tomada de decisão (`if`, `elif`, `else`).
* **Estruturas de Repetição:** Laços de repetição (`for`, `while`).
* **Funções e Listas:** Organização e manipulação de dados.
* **Projetos de Lógica:** Pequenos sistemas que resolvem problemas do cotidiano.

## 💻 Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Ambiente:** VS Code
* **Controle de Versão:** Git & GitHub

## 📝 Como visualizar os códigos
1. Navegue pelas pastas do repositório.
2. Os arquivos possuem a extensão `.py`.
3. Sinta-se à vontade para clonar o repositório e testar os exercícios localmente!
